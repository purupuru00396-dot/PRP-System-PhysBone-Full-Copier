## <span className="updateVersion">v1.1.5</span>（<span className="updateDate">2026/01/01</span>）


### ■ コンポーネントコピー機能の追加 <span className="badgeNew">NEW</span>


従来の PhysBone / Collider に加え、  Animator や Modular Avatar 関連コンポーネントも  まとめてコピーできるようになりました。

コピー対象はフィルター（チェックボックス）で選択可能です。

<span style={{ display: "block", height: "30px" }}></span>

※ メッシュ系 Modular Avatar コンポーネント  

（例：MA Shape Changer）は、環境によって挙動が崩れる可能性があるため、  オプション扱い（初期設定 OFF）としています。



---



### ■ Modular Avatar 互換性の強化 <span className="badgeNew">NEW</span>  

（トグル／衣装切り替え + Shape Changer 自動マッチング）

<span style={{ display: "block", height: "30px" }}></span>


- アバターのルート名やオブジェクト名が異なっていても、  

&nbsp; Merge Animator（トグル／衣装切り替え）のアニメーションパスを  

&nbsp; 自動で調整し、動作しやすくなるよう改善しました。


<span style={{ display: "block", height: "30px" }}></span>

- MA Shape Changer 使用時、  &nbsp; 衣装側アバターが別であっても、  &nbsp; 内部のボーン構造やオブジェクト構成が同一であれば、  

&nbsp; 自動で接続先を検出してマッチングします。



これにより、手動での再接続作業を大幅に削減できます。



---



### ■ マッチングロジックの改善 <span className="badgeNew">NEW</span>



以下のような軽微な差分があっても、  

可能な範囲で近い対象を自動検出してマッチングするよう改善しました。



\- 空白 / `\_` / `-`

\- `.1` / `(Clone)` などのサフィックス違い



---



### ■ Copy Failure Inspector の改善 <span className="badgeNew">NEW</span>



従来の Structure Diagnostic Window を刷新し、  単純な比較ではなく、失敗原因をカテゴリ別に表示します。



\- ボーン名の不一致（例：`.001`）

\- 重複ボーン

\- 階層（パス）の違い

\- 対象オブジェクトの欠落



また、診断結果をテキストとしてコピー可能なレポート形式に整理しました。



---



### ■ ログ保存・リセット挙動の改善 <span className="badgeNew">NEW</span>



- 作業ログが自動でファイルに保存されます  

&nbsp; （例：  &nbsp; `Assets/! PRP System !/Phys Bone Full Copier/Logs/PhysBoneCopyLog.txt`）



- Reset 実行時、  

&nbsp; 今回のバージョンでコピーされたコンポーネントも含めて&nbsp; まとめて初期化されるよう調整しました。



</div>
