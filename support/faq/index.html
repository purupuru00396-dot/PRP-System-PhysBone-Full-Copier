❓ よくある質問（Q&A）

Q1. アバター本体にも使用できますか？​

基本的には可能です。

コピー元とコピー先で ボーン構造が同一であれば、アバター本体に対しても動作します。

ただし、本ツールは衣装・アクセサリー制作を主目的として設計されたツールです。

アバター本体に使用する場合は、必ずコピー後の挙動を確認してください。

Q2. ヘア用 Prefab にも使えますか？​

はい、使用できます。ヘア Prefab についても、ボーン構造が一致していればコピー可能です。

ただし、Collider を含む場合は、Collider が配置されている GameObject 名やRoot Transform の構造によってはコピーされないことがあります。

この場合は、Copy Failure Inspector Tools を使って差分を確認してください。

Q3. PhysBone と Collider のコピーはどのように動作しますか？​

PhysBone Full Copier は、同一のボーン構造を持つ Prefab 間でPhysBone / Collider の設定をコピーします。

以下の条件が重要です。

  • 対象ボーンが対応付けられること

  • Collider の Root Transform や参照先が一致していること

  • 名前や構造に差分がある場合、自動コピーされない、または一部のみコピーされることがあります。

その際は、診断ツールで原因を確認してください。

Q4. Material の設定もコピーできますか？​

はい、可能です。

衣装に設定されている Material を、修正後の FBX や別の衣装にコピーできます。

ただし、Material が正しくコピーされるためには条件があります。

重要なポイント（ここだけ確認してください）

Material は、割り当てられているメッシュ名が一致している場合のみ 正しくコピーされます。

❌ コピーされない例

以下のように、名前が少しでも違う場合はコピーされません。

  • belt → belt.001

見た目が同じでも、メッシュ名が異なると別物として扱われます。

Q5. コピーに失敗した場合はどうすればいいですか？​

コピーに失敗した場合は、まず Copy Failure Inspector Tools を開いてください。

このツールでは、

  • 名前の違い
  • 階層（Hierarchy）の違い
  • 対象が存在しない要素

などを一覧で確認できます。原因を確認した上で、

必要な部分のみ修正することをおすすめします。

Q6. ツールのアップデートはありますか？​

はい。今後も 不具合修正や機能改善を中心にアップデートを予定しています。

最新情報は、BOOTH の販売ページおよび更新履歴をご確認ください。

💡 補足（はじめての方へ）

「コピーできない＝エラー」ではありません。

多くの場合は、名前や構造の違いによる 仕様上の挙動 です。

焦らず、診断結果を一つずつ確認してください。

Q7. マニュアルを見ても解決しない場合は？​

うまくコピーできない原因が分からない場合は、 Log テキスト欄の内容をコピーしてお問い合わせください。

Log テキスト欄をクリックしてから、 Ctrl + C（コピー） / Ctrl + V（貼り付け） が可能です。

できるだけ状況を詳しく書いたうえで、 Log テキストを添付して BOOTH からご連絡ください。

また、Discord でもサポートしています。 必要であれば、下記リンクからご参加ください。
